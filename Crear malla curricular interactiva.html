Crear una malla curricular interactiva desde GitHub es totalmente posible usando herramientas web modernas. Aquí te explico paso a paso cómo hacerlo, desde lo más básico hasta algo visual e interactivo que puedas mostrar como portafolio, proyecto universitario o incluso para una institución.

🔧 PASOS PARA CREAR UNA MALLA CURRICULAR INTERACTIVA EN GITHUB

1. Define la estructura de tu malla curricular

Piensa en:

  • Carreras → Ej: Ingeniería Civil

  • Semestres → Ej: Semestre 1, 2, etc.

  • Asignaturas → Ej: Cálculo I, Física, Ética

  • Requisitos → Ej: Requiere Cálculo I para tomar Cálculo II

Puedes representar esto como un archivo JSON, YAML o directamente en código HTML/JavaScript.

2. Crea un repositorio en GitHub

  1. Ve a github.com y crea una cuenta si no tienes.

  2. Haz clic en New Repository.

  3. Ponle un nombre como malla-curricular-interactiva.

  4. Marca la opción Initialize with README.

3. Crea una web interactiva con HTML + CSS + JavaScript

Puedes hacerlo tú misma o usar una plantilla ya hecha.

OPCIÓN A: Usar una plantilla simple (recomendado si recién comienzas)

  1. Ve a este repositorio de ejemplo: curriculum-map

  2. Haz clic en Fork (arriba a la derecha).

  3. En tu copia del repositorio, haz clic en Settings > Pages.

  4. En "Source", selecciona main y luego save.

  5. ¡Tu malla ahora está publicada en internet! El link será algo como:

    perl
    https://tuusuario.github.io/curriculum-map/

OPCIÓN B: Hacerlo desde cero con HTML

Puedes usar una grilla visual tipo "cards" o "bloques" para representar los ramos y relaciones de prerrequisitos.

Ejemplo básico de HTML:

html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8" />
  <title>Malla Curricular</title>
  <style>
    .asignatura {
      border: 1px solid #ccc;
      padding: 10px;
      margin: 10px;
      border-radius: 8px;
      background-color: #f5f5f5;
    }
    .semestre {
      display: flex;
      flex-direction: column;
    }
  </style>
</head>
<body>
  <h1>Malla Curricular - Administración Pública</h1>
  <div class="semestre">
    <div class="asignatura">Introducción a la Administración</div>
    <div class="asignatura">Microeconomía</div>
    <div class="asignatura">Ética Pública</div>
  </div>
</body>
</html>

4. Agrega interactividad con JavaScript

Puedes usar JS para mostrar relaciones entre asignaturas, requisitos, o información desplegable al hacer clic.

Ejemplo:

js
document.querySelectorAll('.asignatura').forEach(item => {
  item.addEventListener('click', () => {
    alert('Descripción de la asignatura: ' + item.textContent);
  });
});

5. Publica con GitHub Pages

  1. Sube tu código HTML/CSS/JS al repositorio.

  2. Ve a Settings > Pages.

  3. Publica tu sitio desde la rama main y carpeta /root.

  4. ¡Tu malla estará en línea!

6. Mejoras posibles

  • Usar D3.js o GoJS para crear gráficos de prerrequisitos interactivos.

  • Agregar filtros (por semestre, por área).

  • Mostrar estado de aprobación (aprobado, cursando, pendiente).

  • Usar Bootstrap o Tailwind CSS para mejor diseño visual.
